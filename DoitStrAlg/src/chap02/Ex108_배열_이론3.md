###### 65p
### 두 값의 교환
- 임시 변수에 값을 보관
```java
// a와 b의 교환
int temp = a;
a = b;
b = temp;
```
- 배열 안의 두 요소를 교환하는 메서드는 06장(정렬)에서도 자주 사용
```java
static void swap(int[] a, int idx1, int idx2) {
    int temp = a[idx1];
    a[idx1] = a[idx2];
    a[idx2] = temp;
}
```

###### 69p
### cf. 전(후)위형 증가 연산자

```전위형 증가 연산자 ++a```
- 식 전체를 평가하기 전에 피연산자의 값을 증가시킴

```후위형 증가 연산자 a++```
- 식 전체를 평가한 후 피연산자의 값을 증가시킴

```java
a = 3;
b = ++a;
// a = 4, b = 4
a = 3;
b = a++;
// a = 4. b = 3
```
